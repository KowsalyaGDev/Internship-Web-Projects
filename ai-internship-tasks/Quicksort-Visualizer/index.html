About Quicksort

Quicksort is a fast, efficient sorting algorithm that divides the array into smaller sub-arrays and sorts them recursively. It uses a "divide and conquer" approach by selecting a pivot element and partitioning the array around it.

Time Complexity: O(n log n) average, O(n²) worst case

Space Complexity: O(log n)
